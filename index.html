# 🌿 Bamboo Notes > 日常で見つけた英単語を、あなたのものに。 英単語アプリの新しいカタチ。 AIと一緒に「使える英語力」を育てる🌱 --- ## ✨ どんなアプリ？ **Bamboo Notes** は、 日常で出会った英単語を簡単に記録・復習できる **AI搭載の英語学習アプリ**です！ 英単語を入力するだけで、 意味・例文・フィードバックをAIが自動で生成🪄 > 「1分のメモが、あなたの英語力を変える。」 --- ## 🧠 特徴 ### 🌱 シンプルで直感的 - 単語を入力するだけ！ - 意味と例文をAIが自動で生成 - ノート感覚でどんどん追加できる ### 🧩 脳科学×AIで記憶を定着 - 6:00 / 12:00 / 18:00 / 0:00 に更新される英作文チャレンジ - 少しずつ繰り返すことで、無理なく覚えられる - 短時間でサクッと復習できるリズム学習🎵 ### 🐼 学習を見守るキャラクター - 時間帯で見た目が変わる！（朝・昼・夜・深夜） - ホーム画面であなたを応援してくれる🌞🌙 - 単語数に応じてキャラが解放 - ホーム画面に現れるかも。。。 --- ## 🚀 使い方（How to Play） 1. **単語を追加してリスト化** 　👉 見つけた単語をメモ感覚で登録！ 2. **短い会話から意味を推測** 　🤔 自分の理解をAIがチェック！ 3. **1日3回のチャレンジに挑戦！** 　🔥 朝・昼・夜のミニテストで記憶をアップデート！ --- ## 🧩 技術スタック | カテゴリ | 使用技術 | |-----------|-----------| | フロントエンド | Flutter (Dart) | | バックエンド | Firebase Functions (Node.js) | | AIモデル | OpenAI GPT-4o-mini | | データ管理 | Firestore / SharedPreferences | | 広告 | Google AdMob (Rewarded / Interstitial) | | デプロイ | App Store Connect (iOS) | | Webサイト | GitHub Pages | --- ## 🤖 主なAI機能 | 機能 | 内容 | |------|------| | ✍️ `generateDefinition` | 定義＋例文を自動生成 | | 💬 `evaluateSentence` | 英作文をAIがフィードバック | | 🌍 `generateContextualExample` | 文脈に合う例文を生成 | | 🎯 `DailyChallenge` | 1日3回の英作文チャレンジ | --- ## 🌐 関連リンク - 🧾 [プライバシーポリシー]([https://applicationsup12-eng.github.io/privacy-policy](https://smooth-front-52a.notion.site/2821020a4aaa80d5afdcee603263abf8?source=copy_link) - 🎥 [TikTok 公式]([https://www.tiktok.com/@bamboo_notes_app](https://www.tiktok.com/@bamboo.notes?_t=ZS-90lKwYxexpU&_r=1) - 📱 [App Store](https://apps.apple.com/jp/app/bamboo-notes/id6753908452) --- ## 👩‍💻 開発者メモ > “英語をもっと自然に使える世界をつくりたい。” | 項目 | 内容 | |------|------| | 分野 | 教育 × AI × 言語習得 | | Vision | 学ぶことを「楽しい」に変える🌈 | --- 🎋 **Bamboo Notes** > 「単語を“見る”アプリから、“使う”アプリへ。」 © 2025 Bamboo Notes Team
